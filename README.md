# 🏠 Projeto de Modelagem de Processo de Locação de Imóvel

Este projeto tem como objetivo **modelar o processo de locação de imóvel** utilizando a notação **BPMN** (Business Process Model and Notation) e a ferramenta **Bizagi Modeler**. O foco é otimizar o fluxo de trabalho, automatizar validações e melhorar a comunicação entre os setores envolvidos (corretor, gestor e recepção).

## 🎯 Objetivos do Projeto

- **Otimizar o processo de locação de imóveis** desde o preenchimento dos dados do cliente até a entrega das chaves.  
- **Automatizar a validação** da disponibilidade de imóveis e a **análise de crédito** do cliente, com base em integrações externas.  
- **Padronizar as etapas do processo**, garantindo clareza nas responsabilidades e ações de cada ator (corretor, gestor, recepção).  
- **Melhorar a comunicação interna**, minimizando erros e atrasos no processo de locação.  
- **Aumentar a satisfação do cliente**, tornando o processo mais ágil e eficiente.  

## 🏗️ Estrutura do Processo

O processo de locação é dividido em três **raias** (pools):

1. **Corretor**: Responsável pela coleta de dados do cliente, visita ao imóvel, coleta de documentos e envio para aprovação.
2. **Gestor**: Responsável pela análise de crédito do cliente e aprovação do processo de locação.
3. **Recepção**: Responsável pela finalização administrativa, incluindo a entrega das chaves ao cliente.

## ⚙️ Funcionalidades

- **Preenchimento dos dados do cliente**: Nome, CPF, forma de contato, imóvel desejado.  
- **Validação de disponibilidade de imóvel**: Verificação se o imóvel está disponível.  
- **Visita ao imóvel e feedback do cliente**: Registro das impressões do cliente.  
- **Coleta de documentos**: Garantir que todos os documentos necessários sejam coletados.  
- **Análise de crédito automatizada**: Integração com sistemas de verificação de crédito.  
- **Submissão de contrato e finalização**: Envio de contrato para aprovação final.  
- **Entrega das chaves**: Finalização do processo com a entrega das chaves ao cliente.  

## 🚀 Como Rodar

1. **Bizagi Modeler**:
   - Abra o arquivo `.bpm` no **Bizagi Modeler**.
   - O modelo BPMN estará carregado com as etapas e fluxos já configurados.
   
2. **Requisitos**:
   - Não há requisitos específicos para a execução, pois o projeto está focado na modelagem do processo de locação.

## 📈 Possíveis Melhorias Futuras

- **Criação de KPIs**: Implementação de indicadores de desempenho como tempo de aprovação e taxa de aprovação de crédito.  
- **Integração com sistemas de pagamento**: Facilitar a transação do pagamento de aluguel.  
- **Otimização da análise de crédito**: Implementação de modelos preditivos para análise mais assertiva.
